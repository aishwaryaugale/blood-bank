## ÔøΩ Quick Start

### Prerequisites
- Python 3.x installed
- Modern web browser
- Same WiFi network for mobile sharing

### Installation & Setup

1. **Clone or download the project**
   ```bash
   # Navigate to your project folder
   cd d:\Blood
   ```

2. **Start the development server**
   ```bash
   python server.py
   ```

3. **Access the application**
   - **Local URL:** `http://localhost:8000` (for your computer)
   - **Network URL:** `http://192.168.1.XXX:8000` (share this with friends)

### Mobile Preview Setup

1. **Share with friends:**
   - Make sure everyone's on the same WiFi network
   - Share the Network URL shown in terminal
   - Friends open the URL in their phone browsers

2. **Mobile features tested:**
   - Responsive design for all screen sizes
   - Touch interactions and gestures
   - Blood type animations
   - Login functionality
   - Donor search system
   - Dark/light mode toggle
   - Admin dashboard with charts
